# Sales-Performance-Customer-Insights
End-to-end data analytics project using SQL, Python, and Power BI

##  Project Overview
This project analyzes sales data to identify profit drivers, loss-making segments, and regional performance using SQL, Python, and Power BI.

##  Business Problem
Despite strong sales volume, the business experiences inconsistent profitability due to discounts, category-level losses, and regional variation.

## Key Business Questions
- Which categories and regions are most profitable?
- How do discounts impact profit?
- Are high sales always associated with high profit?
- Which category-region combinations are loss-making?

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (Microsoft SQL Server)
- Power BI
- GitHub

##  Project Workflow
1. Data Cleaning (Python & SQL)
2. Exploratory Data Analysis (Univariate, Bivariate, Trivariate)
3. Feature Engineering
4. Dashboard Creation (Power BI)
5. Business Insights & Recommendations

##  Key Insights
- Technology category is consistently profitable
- Furniture shows losses at higher discounts
- West region outperforms other regions
- High discounts negatively impact profit

##  Conclusion
The analysis highlights key profit drivers and risk areas, enabling data-driven decision-making for pricing and regional strategies.
